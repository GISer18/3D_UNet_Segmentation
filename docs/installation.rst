============
Installation
============


Stable release
--------------

To install 3D_Segmentation, run this command in your terminal:

.. code-block:: console

    $ pip install segmentation_3D

This is the preferred method to install 3D_Segmentation, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for 3D_Segmentation can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/lhw610/segmentation_3D

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/lhw610/segmentation_3D/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/lhw610/segmentation_3D
.. _tarball: https://github.com/lhw610/segmentation_3D/tarball/master
